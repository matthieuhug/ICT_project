CentOS Installation Guide

Introduction

CentOS (Community ENTerprise Operating System) is a free and open-source Linux distribution based on Red Hat Enterprise Linux (RHEL). It's ideal for servers and development environments. This guide will walk you through the process of installing CentOS on your machine.

Step 1: Download CentOS

To install CentOS, you first need to download the installation ISO file. Follow these steps:

  • Go to the official CentOS website: https://www.centos.org/download/
  • Select the version of CentOS you want to install (CentOS 7 or CentOS Stream recommended).
  • Download the ISO file to your computer.

Step 2: Create a Bootable USB Drive

You will need a USB drive with at least 4GB of storage to create a bootable CentOS installation media. Here's how to do it:

  • Download a tool like Rufus (for Windows) or use the "Startup Disk Creator" (for Ubuntu).
  • Insert the USB drive into your computer.
  • Open the tool (e.g., Rufus or Startup Disk Creator) and select the downloaded CentOS ISO file.
  • Choose the USB drive as the target disk and click "Start" to create the bootable media.

Step 3: Boot from USB

Once your bootable USB drive is ready, it's time to boot from it:

  • Insert the bootable USB drive into the machine where you want to install CentOS.
  • Reboot the machine and enter the BIOS or UEFI settings (usually by pressing F2, F10, ESC, or DEL during startup).
  • In the BIOS/UEFI settings, change the boot order to boot from the USB drive first.
  • Save the changes and exit the BIOS/UEFI settings. The machine will reboot and boot from the USB drive.

Step 4: Install CentOS

Now that you're booted into the CentOS installation environment, follow these steps:

  • After booting, you should see the CentOS welcome screen. Select "Install CentOS" and press Enter.
  • Choose your language and click "Continue".
  • Select your keyboard layout and click "Continue".
  • For most users, you can choose the default settings (like "Automatically partition the disk" for the disk selection).
  • Set your time zone and click "Done".
  • Click "Begin Installation" to start the installation process.
  • During the installation, you'll be prompted to create a root password and a user account. Fill in these details as required.
  • Once the installation is complete, click "Reboot" to restart the system.

Step 5: Finalizing the Installation

After the installation process is complete, follow these steps to finalize the setup:

  • Remove the installation USB drive when prompted and press Enter to reboot the system.
  • Your system will now boot into CentOS, where you can log in using the root password or the user account you created during installation.

Step 6: Post-Installation Steps

After CentOS is installed, here are some post-installation steps you may want to consider:

  • Update your system to make sure you have the latest security patches and software updates. Open a terminal and run the following command:
    • sudo yum update -y
  • Install additional software. For example, you can install software like VLC or GIMP using the following command:
    • sudo yum install vlc gimp -y
  • Configure system settings, including enabling firewall rules, SELinux configuration, or network settings if necessary.
